Resumo: Desenvolver um gateway que interpõe dispositivos entre o usuário e sistemas corporativos de IA, que aplicará:
- Autenticação contínua
- Tunelamento* criptografado de ponta a ponta
- DLP (Data Loss Prevention) no nível da API

*No contexto de redes de computadores, o tunelamento consiste em criar um “túnel” virtual entre dois pontos de uma rede, permitindo que pacotes de dados sejam enviados de forma segura, mesmo quando os protocolos originais não são compatíveis com a rede intermediária. O processo envolve encapsular um pacote dentro de outro, de modo que o pacote original fique protegido e possa atravessar firewalls ou redes públicas sem ser interceptado.
